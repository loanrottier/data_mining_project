# data_mining_project
Academic project of Datamining about anomaly detection with data depth

Subject C: Outlier Detection with Data Depth
Ressources:
Article: Mozharovskyi, P., & Valla, R. (2025). Anomaly detection using data depth: multivariate
case. International Journal of Data Science and Analytics, 1-26. (link)
Code: data-depth package :

[package website](https://data-depth.github.io/)
[pypi link](https://pypi.org/project/data-depth/)


Using the reference article, present data depth and its application to anomaly detection. Groups
of 2 students are required to study and present two depth notions of their choice, while groups
of 3 students must present three depth notions of their choice.
